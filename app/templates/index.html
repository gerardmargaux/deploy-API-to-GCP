Summarize uses a model developed by Hugging Face to summarize a text very quickly. So you can copy and paste a very long article you don't have time to read and summarize it in a few seconds to get the main idea.
Summarize is a personal project that aims to deploy an API to Google Cloud Platform (GCP), while provisioning the required infrastructure with Terraform.
